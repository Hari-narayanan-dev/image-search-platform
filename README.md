## 📖 Overview

Image Search Platform is a responsive web application that enables users to search millions of high-quality images through the Unsplash API. The application provides a clean interface with instant search results and pagination for browsing large image collections.

This project serves as the **Version 1.0** foundation and will evolve into a full-stack AI-powered Digital Asset Management platform in future releases.

---

## ✨ Features

- 🔍 Search images using keywords
- 📷 Browse high-quality images from Unsplash
- 📄 Pagination support
- ⚡ Fast search experience
- 📱 Responsive user interface

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/image-search-platform.git

cd image-search-platform
```

### Install dependencies

```bash
npm install
```

### Configure API Key

Create a `.env` file in the project root.

```env
VITE_UNSPLASH_ACCESS_KEY=YOUR_UNSPLASH_ACCESS_KEY
```

---

### Run the application

```bash
npm run dev
```

Open your browser:

```
http://localhost:5173
```
